GLOBAL SYNLOGIA - KOLORY BRANDOWE
=====================================

OFICJALNE KOLORY FIRMY:
-----------------------

1. NIEBIESKI (Główny kolor brandowy)
   - HEX: #124f9e
   - Zastosowanie: Główny kolor akcji, nagłówki, linki LinkedIn, ikony główne
   - Hover: #0f3d7a (ciemniejszy odcień)

2. CZERWONY (Drugi kolor brandowy)
   - HEX: #de244b
   - Zastosowanie: Akcenty, wyróżnienia, ikony drugorzędne, call-to-action

ZASTOSOWANIE W KODZIE:
---------------------

CSS/Tailwind klasy:
- text-[#124f9e] - tekst niebieski
- text-[#de244b] - tekst czerwony
- bg-[#124f9e] - tło niebieskie
- bg-[#de244b] - tło czerwone
- border-[#124f9e] - border niebieski
- border-[#de244b] - border czerwony

Hover states:
- hover:text-[#0f3d7a] - hover dla niebieskiego
- hover:bg-[#0f3d7a] - hover dla tła niebieskiego

ZASADY STOSOWANIA:
-----------------
- TYLKO te dwa kolory w całej stronie
- NIE używać innych kolorów (zielony, fioletowy, pomarańczowy)
- Niebieski jako kolor główny
- Czerwony jako kolor akcyjny/drugorzędny
- Zachować kontrast z białym tłem
- Użycie szarości tylko dla tekstu pomocniczego
